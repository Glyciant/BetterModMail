Welcome to BetterModMail

This page will hopefully help to explain how to use this site effectively.

Viewing Messages

To view messages from the subreddit, select the All Mail or My Mail dropdown box, and choose a filter. If you are new to the site, you will want to use All in All Mail

Selecting My Mail filters the messages to only show the ones assigned to you. The sub-filters sort the messages by their status in addition to this.

Applying Tags to Messages

Tags can be used for multiple purposes, mostly for TL;DRs and CTRL + F keywords. To add tags to a message, use the following method:

  1. Go to the messages table.
  2. Find the action column of the message you want to add tags to.
  3. Click the first button. A key of the buttons is at the top of the page.
  4. In the popup that appears, enter the tags you wish to add or edit.
  5. Click Save and Close
Assigning User to a Message

Assigning a user allows message to be filtered easier. You can assign a user by doing the following:

  1. Go to the messages table.
  2. Find the action column of the message you want to add tags to.
  3. Click the second button. A key of the buttons is at the top of the page.
  4. In the popup that appears, select the moderator you wish to assign.
  5. Click Save and Close
Changing a Message's Status

Changing a messages status allows message's to be filtered and managed easier. You can do this with the following method:

  1. Go to the messages table.
  2. Find the action column of the message you want to add tags to.
  3. Click the third button. A key of the buttons is at the top of the page.
Replying to a Message

BetterModMail has a built-in system for replying to messages. You can use it with the following method:

  1. Go to the messages table.
  2. Find the action column of the message you want to add tags to.
  3. Click the fifth button. A key of the buttons is at the top of the page.
  4. In the popup that appears, type the message you wish to send. You can find a drop-down box above the textbox with some pre-made responses.
  5. Click Send and Close
Creating a Message

Creating a message allows you to start a modmail conversation, that will then appear on the table. To do this, you need to:

  1. Click Compose Message on the navigation bar.
  2. Fill in all the fields.
  3. Click the Send Message button.
Muting a User

Muting a user prevents their messages from showing, no matter what filters are set. Unlike on reddit, this is indefinate and the user is not alerted. You can mute a user by:

  1. Click Muted Users on the navigation bar.
  2. Type the username and a reason to mute them.
  3. Click the Mute User button.
